# Claude Skills

Custom Claude Code skill files for Aspora projects.

## Skills

| Skill | Description |
|-------|-------------|
| `excel-research-agent` | Deep research agent that fills missing spreadsheet data using web search and AI knowledge |
| `excel-research-agent-v2` | V2 with improved research depth and formatting |
| `excel-research-agent-v3` | V3 with skeleton expansion — treats templates as minimum structure, not limits |
| `excel-research-agent-v4` | V4 with mandatory discovery pass and completeness audit |
| `project-init` | Setup wizard for new Aspora project skill sets (generates config, update, context, and run history files) |
| `debug` | Structured debugging workflow |
| `feature` | Feature implementation workflow with branching and tests |
| `review` | Code review checklist (security, errors, performance, tests) |

## Usage

Copy files from `commands/` to `~/.claude/commands/` to use as Claude Code slash commands.
